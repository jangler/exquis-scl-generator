Isomorphic note layout heatmaps

These interactive plots score isomorphic MIDI controller note layouts on LCJI approximation. Plots are organized by step sizes, measured in cents.

Specifically, a layout's score is the sum of inverse Tenney heights of 16-integer-limit intervals scaled according to each interval's best approximation on the keyboard if the center key is 1/1. The penalty operates on the square of the error, with 15Â¢ or more counting as a complete miss.

There are two Exquis options because only a 39-note subset of the Exquis pads can be remapped until support for full-keyboard remapping is implemented.

For the Launchpad and LinnStrument, there is no true center pad. The nearest-to-center pad in the bottom-left quadrant is used as 1/1.
