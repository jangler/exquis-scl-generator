Exquis .scl generator

This webpage generates isomorphic microtonal layouts for the Exquis using Scala scales. Since the Exquis does not currently support remapping pads, the outermost "columns" of pads are duplicate notes.

Intervals can be specified as ratios (ex. 5/4), cents values (ex. 386.3), or octave fractions (ex. 10\31).
